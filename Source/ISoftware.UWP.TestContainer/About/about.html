Embedded HTTP server

The Embedded HTTP server is a very simple HTTP server that can be embedded in a Universal Windows Platform application. It can be used to communicate with the application, primarily to control it, using the HTTP protocol. The application protocol being HTTP, makes it possible to use any of the many HTTP client libraries to communicate with the UWP application. The Embedded HTTP server can also be used to provide an Admin interface for the device, allowing the device to be configured and its status reported.
